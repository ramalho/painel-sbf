Notas escritas por Ernesto S. Junior:

1 - pino geral : é uma porta do fundo para inutilizar o painel para manutenção, mas que ficara ligado direto com um jumper.

2 - quando ligar a maquete as 3 chaves ficarão piscando em estado de espera da seleção de um fonte de energia externa, analogico, digital, wifi.

3 - para selecionar uma fonte, operador devera manter apertado um dos tres botoes de selecao por 5s

4 - quando selecionado a fonte o botão habilita fica disponivel para ligar a linha e quando ligada serve tambem de parada de emergencia.

5 - por ultimo quando habilitado se houver um curto circuito o botão habilita desarme e fica piscando, ate que o curto seja resolvido

6 - quando o curto e resolvido, a selecao de fonte e' mantida, mas o botao habilita continua desligado

7 - para liberar a linha para outro operador, o operador aciona de novo o botao da fonte que esta selecionada. dai ela volta pro estado 2
